# Tech Product Revenue Forecast Model

## Overview
This project analyzes historical revenue trends across major technology companies and builds a simple forecasting model to project future Google revenue growth.

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Project Goals
- Compare historical revenue growth across major technology companies
- Visualize financial trends over time
- Build a simple forecasting model using linear regression
- Generate insights relevant to financial planning and analysis

## Files
- `data/company_revenue.csv` — historical revenue dataset
- `notebooks/revenue_forecast.ipynb` — analysis notebook with charts and forecasting model

## Key Outputs
- Revenue comparison chart for Apple, Google, and Samsung
- Google revenue forecast for 2025–2027
- Visualization of historical and projected revenue trends
